# MemoryDeck

MemoryDeck is a full-stack flashcard learning application that allows users
to create decks, manage cards, practice efficiently, and track learning progress.

## Features

* User authentication
* Deck and card management
* Practice sessions with real-time feedback
* Practice history and statistics
* Clean and consistent learning-focused UI

## Tech Stack

* Frontend: Angular
* Backend: ASP.NET Core
* Database: Entity Framework Core
